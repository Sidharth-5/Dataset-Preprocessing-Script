# Bhojpuri Morisien – Kreol Morisien Master Dataset

This repository contains the preprocessing scripts used to clean and standardize datasets.

## Contents
- preprocessing_script.py – Data cleaning pipeline

## Preprocessing Steps
- Lowercasing text
- Unicode normalization
- Removal of non-alphanumeric characters
- Whitespace standardization
- Duplicate removal
- Filtering incomplete entries

## Requirements
- Python 3.x
- pandas
